# Tech Vocabulary (技术词汇)

> **Objetivo:** 200+ palabras técnicas en mandarín — programación, AI, ingeniería. Suficiente para leer documentación técnica en chino y entender conversaciones de ingeniería.

## Objetivos de aprendizaje

- Memorizar 50 palabras base: programación, estructuras de datos, AI/ML
- Reconocer patrones de formación de palabras técnicas (calcos del inglés, compuestos semánticos)
- Leer un README técnico corto en chino sin diccionario (>80% comprensión)
- Producir frases técnicas simples: describir qué hace una función, nombrar componentes de un sistema
- Usar Anki/Pleco SRS para mantener retención >90% a 30 días

## Conceptos clave

### Categorías de vocabulario técnico

| Categoría | Ejemplos | Notas |
|-----------|----------|-------|
| Programación | 代码 (code), 函数 (function), 变量 (variable), 循环 (loop) | Muchos son calcos semánticos del inglés |
| Estructuras de datos | 数组 (array), 链表 (linked list), 哈希表 (hash table), 树 (tree) | Compuestos descriptivos: 链=cadena + 表=tabla |
| AI/ML | 机器学习 (machine learning), 神经网络 (neural network), 训练 (training), 模型 (model) | Términos estandarizados por la industria China |
| Ingeniería | 系统 (system), 架构 (architecture), 部署 (deploy), 性能 (performance) | Alta frecuencia en documentación |
| Herramientas | 编辑器 (editor), 终端 (terminal), 调试器 (debugger), 编译器 (compiler) | 器 = sufijo para herramienta/instrumento |

### Patrones de formación

```
Calco fonético:    博客 (bókè) = blog, 黑客 (hēikè) = hacker
Calco semántico:   鼠标 (shǔbiāo) = mouse (ratón + marcador)
Compuesto nativo:  电脑 (diànnǎo) = computer (eléctrico + cerebro)
Préstamo híbrido:  Python → Python (escrito igual, pronunciado pàisēn)
```

### Falsos amigos y trampas

| Chino | Parece que significa | Significa realmente |
|-------|---------------------|---------------------|
| 函数 (hánshù) | "número de carta" | Función (数学函数 = math function) |
| 对象 (duìxiàng) | "par de elefantes" | Objeto (面向对象 = object-oriented) |
| 指针 (zhǐzhēn) | "aguja de papel" | Puntero (指针地址 = pointer address) |
| 线程 (xiànchéng) | "ruta de hilo" | Thread (线程安全 = thread-safe) |

**Semana 1 — 50 palabras base + patrones**

- Día 13: Leer documentación de PyTorch o TensorFlow en chino — subrayar palabras nuevas
